OSBDM directory description:
============================

1) doc:			Documentation that needs to be in OSBDM package distribution.
2) gdi: 		OSBDM GDI driver on host which interfaces with CodeWarrior debugger.
3) osbdm:		Host OSBDM USB driver, host USB driver, JM60 USB/OSBDM firmware, and 
			USB command set tester.
